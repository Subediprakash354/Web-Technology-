Use of <p> tag

The <p> tag is used to write paragraphs in HTML. It automatically adds some space before and after the text. For example, this is a simple paragraph created using the <p> tag.

Use of <pre> tag

The <pre> tag preserves
    spaces and line breaks
exactly as written in the HTML code.

For example:
    Name: Ram
    Class: 12
    Subject: HTML
